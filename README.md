## Set Up
### install rust on wsl: 
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

### Install cargo
Sudo apt-get update
sudo apt install cargo

## Create a Rust Template
cargo new transaction-decoder
cd transaction-decoder

## Run it
cargo run
